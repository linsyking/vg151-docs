Ex4. Basic Programming#

\[\begin{split}\begin{cases} u_0 = a\\ u_{i+1} = \begin{cases} \frac{1}{2}u_i, \quad &\text{if $u_i$ is even}\\ 3_{u_i}+1, &\text{if $u_i$ is odd} \end{cases} \end{cases}\end{split}\]

  • Basic logic questions

  • Think about Fibonacci sequence. How to find the \(n\)-th Fibonacci number?

Hint

How to optimize the following recursion function?

long findn(long a) {
    if (a == 1) return 0;
    if (a % 2) {
        return findn(3 * a + 1) + 1;
    } else {
        return findn(a >> 1) + 1;
    }
}

Solution. We can use tail recursion to optimize it.

long findn(long a, long cnt) {
    if (a == 1) return cnt;
    if (a % 2) {
        return findn(3 * a + 1, cnt + 1);
    } else {
        return findn(a >> 1, cnt + 1);
    }
}

ex5.cpp#
class Grade {
private:
    static const char GRADES[] = {...};
    // Or use string
    char                  ltr;
    int                   prct;

public:
    void setPrct(int prct) {
        this->prct = prct;
        this->ltr  = GRADES[prct / 10];
    }
    void setLtr(char ltr) {
        this->ltr  = ltr;
        this->prct = 100 - (ltr -'A') * 10 - 5;
    }
    void print() { std::cout << "Grade:" << this->prct << "->" << this->ltr << std::endl; }

Ex1. Containers#

  • Basic C++ Containers library & Data structure

    Make full use of cppreference.

  • Parameter T inside <> is called template parameter, deduced when compilation

  • std::array<T, N>: static contiguous array

  • std::vector<T>: dynamic contiguous array

  • std::stack<T>: adapts a container to provide stack (LIFO data structure)

  • std::queue<T>: adapts a container to provide queue (FIFO data structure)

You should know the basic operations of those containers.
